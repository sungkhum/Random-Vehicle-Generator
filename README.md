# Vehicle Roulette

Filter thousands of open-source vehicles and pull a random result from across air, land, road, rail, sea, and space.

## Included categories

- Fighter jets
- Bombers
- Attack aircraft
- Military transports
- Airliners
- Helicopters
- Attack helicopters
- Tanks
- Tank destroyers
- Armored personnel carriers
- Infantry fighting vehicles
- Recon vehicles
- MRAPs
- Armored cars
- Rocket artillery
- Self-propelled artillery
- Amphibious assault vehicles
- Sports cars
- Supercars
- Race cars
- Muscle cars
- Pickup trucks
- SUVs
- Buses
- Locomotives
- High-speed trains
- Sailboats
- Submarines
- Aircraft carriers
- Spacecraft

## Local use

```bash
npm run build:data
npm run start
```

Then open `http://localhost:4173`.

## Data sources

- Wikidata structured data (CC0)
- Wikipedia vehicle list and category pages (CC BY-SA)

The generated dataset is committed in `data/vehicles.json` and `data/vehicles.js`.
